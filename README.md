# melbourne-house-prices-ml
Predicting Melbourne house prices using ML models (Linear Regression, Random Forest, Gradient Boosting).

## Цель проекта
Предсказание стоимости домов в Мельбурне с использованием ML моделей:  
Linear Regression, Decision Tree, Random Forest, Gradient Boosting.

## Данные
Данные взяты из открытого датасета Melbourne housing dataset.

## Этапы работы
1. EDA и очистка данных
2. Обработка пропусков и выбросов
3. Кодирование категориальных признаков
4. Масштабирование числовых признаков
5. Обучение моделей:
   - Linear Regression
   - Decision Tree
   - Random Forest
   - Gradient Boosting / XGBoost
6. Сравнение моделей по метрикам MAE и RMSE
7. Визуализация важности признаков
